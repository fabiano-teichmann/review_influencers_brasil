Cada linha do CSV contém os campos `nickname` e `review`.
Sua tarefa é analisar o conteúdo do campo `review` e, se necessário, considerar o `nickname` para identificar e classificar aspectos da análise em três categorias:

1. **strengths**: Identifique pontos positivos no review. Classifique com uma ou mais das seguintes tags, se presentes:
   - friendly
   - professional
   - engaged
   - collaborative
   - good return

2. **weaknesses**: Identifique pontos negativos no review. Classifique com uma ou mais das seguintes tags, se presentes:
   - unprofessional
   - bad return
   - not engaged
   - not collaborative

3. **complaint**: Identifique reclamações no review. Classifique com uma ou mais das seguintes tags, se presentes:
   - personality
   - agenda
   - results
   - follow brief
   - advisory team
   - price

**Requisitos da Saída:**
- Para cada linha de entrada, retorne uma linha de saída com as colunas: `nickname`, `strengths`, `weaknesses`, `complaint`.
- Utilize vírgula como delimitador, com cabeçalhos incluídos na primeira linha.
- Seja conciso, mas preciso, na atribuição das tags, listando apenas as tags que se aplicam. Caso não haja identificação de uma categoria, deixe o campo vazio.
